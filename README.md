# gome - a go game framework
[![GoDoc](https://godoc.org/github.com/GomeBox/gome?status.svg)](https://godoc.org/github.com/GomeBox/gome) [![Build Status](https://travis-ci.com/GomeBox/gome.svg?branch=main)](https://travis-ci.com/GomeBox/gome) [![Go Report Card](https://goreportcard.com/badge/github.com/GomeBox/gome)](https://goreportcard.com/report/github.com/GomeBox/gome)

gome is a lightweight framework to develop 2D games with go. It focuses on usability.

**It is still in a very early stage and not yet ready to be used**
